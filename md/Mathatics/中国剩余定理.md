```math
给定一段序列m_1、m_2、... 、m_k 满足两两互质

\begin{cases}
x \equiv a_1(mod \ m_1) \\
x \equiv a_2(mod \ m_2)
... \\
x \equiv a_k(mod \ m_k)
\end{cases}
```
```math
M = \prod_{i = 1}^k m_i 

M_i = \frac M m_i ,
M_i^{-1} 表示M_i模m_i的逆
```
```math
通解：
x = \sum_{i = 1}^k a_i \times M_i \times M_i^{-1}

```
